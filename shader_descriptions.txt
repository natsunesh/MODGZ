# Описание шейдера v1

Этот шейдер создает анимацию с использованием сложных геометрических форм и эффектов освещения. Он включает в себя:

1. **Генерация шума**: Используется для создания эффекта помех и анимации.
2. **Моделирование**: Создаются модели с использованием многогранников (додекаэдр и икосаэдр).
3. **Освещение**: Включает мягкие тени, рассеянное освещение и отражения.
4. **Глитч-эффекты**: Включает эффекты, такие как перемещение блоков, статические помехи и заморозка времени.

# Описание шейдера v2

Этот шейдер создает эффект статичного шума с анимацией. Он включает в себя:

1. **Генерация шума**: Используется симплексный шум для создания анимации.
2. **Эффекты помех**: Включает смещение строк и случайные помехи.
3. **Линейный паттерн**: Применяется линейный паттерн каждые 4 пикселя.
4. **Смещение цветовых каналов**: Смещает зеленый и синий каналы для создания эффекта статичного шума.
